# ✅ クロコ（Claude Code）インストール チェックシート
**【初心者向け】オフ会当日用**

---

## はじめに（読むだけ）

- [ ] 本日の目標：「クロコ」をPCにインストールして動かす
- [ ] 困ったらすぐ手を挙げる（一人で悩まない）
- [ ] スタッフにPCの画面を見せてOK（秘密のファイルは事前に閉じておく）

---

## ■ ステップ0：事前チェック（準備編）

### 環境確認

- [ ] インターネットに接続されている
- [ ] PCの充電が十分ある（またはACアダプタを接続した）
- [ ] ストレージの空き容量が **5GB以上** ある
- [ ] OSのバージョンを確認した
  - Mac：Apple メニュー → 「このMacについて」でバージョン確認
  - Windows：スタート → 設定 → システム → バージョン情報で確認
- [ ] （Macのみ）Apple Silicon（M1/M2/M3/M4）か Intel か確認した

---

## ■ ステップ1：スクリーンショットの練習

**目的：困ったときにスタッフへ画面共有できるようにする**

### Mac

- [ ] **全画面**：`Command ＋ Shift ＋ 3` を押して撮影できた
- [ ] **範囲選択**：`Command ＋ Shift ＋ 4` を押して選択撮影できた
- [ ] 撮影した画像がデスクトップに保存されているのを確認した

### Windows

- [ ] **全画面**：`PrintScreen`（または `Fn + PrintScreen`）で撮影できた
- [ ] **切り取り**：`Windows ＋ Shift ＋ S` でスニッピングツールが開いた
- [ ] 撮影した画像をどこかに貼り付けて確認できた（ペイント等）

---

## ■ ステップ2：黒い画面を開く（ターミナル / PowerShell）

### Mac：ターミナルの起動

- [ ] `Command ＋ Space` を押して Spotlight を開いた
- [ ] 「ターミナル」と入力して Enter を押した
- [ ] 黒（または白）の画面が開いた
- [ ] 文字が入力できる状態になっている（カーソルが点滅している）

### Windows：PowerShell の起動

- [ ] スタートボタンを右クリックした
- [ ] 「Windows PowerShell」または「ターミナル」を選んで開いた
- [ ] 青い画面が開いた
- [ ] 文字が入力できる状態になっている（カーソルが点滅している）

---

## ■ ステップ3：Git（ギット）のインストール

**目的：クロコが内部で使うツールを先に入れる**

### 確認コマンドを実行する

- [ ] ターミナル（PowerShell）に以下を入力して Enter を押した

  ```
  git --version
  ```

- [ ] バージョン番号が表示された（例：`git version 2.xx.x`）→ **インストール済み、スキップ可**
- [ ] 「command not found」等のエラーが出た → **以下のインストール手順へ**

### Git がない場合のインストール

**Mac（Homebrew を使う場合）**

- [ ] 以下を実行した

  ```
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```

- [ ] Homebrew のインストールが完了した
- [ ] `brew install git` を実行した
- [ ] 再度 `git --version` でバージョンが表示された

**Windows**

- [ ] https://git-scm.com/ にアクセスした
- [ ] 「Download for Windows」からインストーラーをダウンロードした
- [ ] インストーラーを実行して「Next」を押し続けてインストールした
- [ ] PowerShell を**閉じて開き直した**
- [ ] `git --version` でバージョンが表示された

---

## ■ ステップ4：クロコ本体のダウンロード

- [ ] 公式サイト `https://claude.ai/download` にアクセスした
- [ ] 自分のOSに合ったインストーラーを選んだ
  - Mac（Apple Silicon / M1以降）→ **macOS (Apple Silicon)**
  - Mac（Intel）→ **macOS (Intel)**
  - Windows → **Windows**
- [ ] ダウンロードが完了した（ファイルがダウンロードフォルダにある）

---

## ■ ステップ5：クロコのインストールと起動

### Mac

- [ ] ダウンロードした `.dmg` ファイルをダブルクリックした
- [ ] アプリをアプリケーションフォルダにドラッグした
- [ ] アプリケーションフォルダから「Claude」を起動した
- [ ] 初回起動時のセキュリティ警告が出た場合：システム設定 → プライバシーとセキュリティ → 「このまま開く」を選んだ

### Windows

- [ ] ダウンロードした `.exe` ファイルをダブルクリックした
- [ ] 「このアプリがデバイスに変更を加えることを許可しますか？」→ 「はい」を押した
- [ ] インストールが完了した
- [ ] スタートメニューに「Claude」が追加された

### 起動確認

- [ ] Claude（クロコ）が起動した
- [ ] Anthropic アカウントでログインした（アカウントがない場合は新規作成）
- [ ] ホーム画面が表示された

### 便利な起動ショートカットの設定

- [ ] デスクトップにショートカット（エイリアス）を作成した
  - Mac：アプリを右クリック → 「エイリアスを作成」
  - Windows：アプリを右クリック → 「ショートカットの作成」

---

## ■ ステップ6：ショートカットと動作確認

### 基本操作の確認

- [ ] クロコに「こんにちは」と話しかけて返事が来た
- [ ] **チャットモード**（通常の会話）が使えることを確認した
- [ ] **エージェントモード**（ファイル操作・コード実行などができるモード）に切り替えた

### モード切り替えの確認

- [ ] 入力欄の左にあるモード切り替えボタンを確認した
- [ ] エージェントモードで簡単な作業（例：「デスクトップにtest.txtを作って」）を試した

### Windows のトラブルシューティング（該当する場合のみ）

- [ ] Windowsセキュリティ（ウイルス対策）がブロックしていないか確認した
- [ ] 管理者権限で実行してみた（右クリック → 「管理者として実行」）
- [ ] 解決しない場合はスタッフに相談した

---

## ■ ステップ7：後片付け（重要！）

**インストールに使ったファイルを消して、PCをスッキリさせる**

- [ ] ダウンロードフォルダを開いた
- [ ] 以下のファイルを削除した（ゴミ箱へ）
  - Mac：`Claude-xxx.dmg`
  - Windows：`Claude-xxx-Setup.exe`
  - Git のインストーラー（Windowsの場合）
- [ ] ゴミ箱を空にした
- [ ] `.dmg` ファイルをマウント（開いた）した場合、デスクトップのディスクアイコンを取り出した（Macのみ）

---

## ■ ステップ8：コストと制限の理解

### 料金の確認

- [ ] クロコの利用プランを確認した
  - 無料プラン：1日の使用量に上限あり
  - Pro プラン（$20/月）：より多く使える
- [ ] 今日の残り使用量を確認する方法を覚えた（画面左下などに表示）

### 節約術

- [ ] 長い会話は新しいチャットを始めた方がトークン節約になることを覚えた
- [ ] 「不要なファイルを読み込ませない」ことを意識する
- [ ] エージェントモードはチャットモードより消費が多いことを把握した

### チャージ（有料プランへの切り替え）

- [ ] 公式サイト（https://claude.ai）からプラン変更できることを確認した
- [ ] クレジットカードによる支払いが必要なことを把握した
- [ ] 不明な場合はスタッフか公式サポートに相談することを覚えた
